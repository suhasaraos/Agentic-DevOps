# Placeholder Image

This folder is for your presentation images.

Add your images here and reference them in slides like:

```html
<img src="images/your-image.jpg" alt="Description">
```

Or for slideshows:

```html
<div class="slideshow-container">
    <div class="slideshow-image active">
        <img src="images/slide1.jpg" alt="First image">
    </div>
    <div class="slideshow-image">
        <img src="images/slide2.jpg" alt="Second image">
    </div>
</div>
```
